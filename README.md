Machine Learning pipeline made to automatically scrape customer reviews, clean raw text data, train a Logistic Regression classifier for sentiment analysis, and visualize data trends. 

first install dependencies:
"pip install -r requirements.txt"
than playwright drivers:
"playwright install"
or:
"playwright install chromium"
for minimal setup 


KEYWORDS
Playwright, ML, Regression Model, Scraping, Sentiment Analysis, NLP, Python, Text Preprocessing, Data Visualization, Pandas, WordCloud
